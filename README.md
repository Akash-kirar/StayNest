🏡 StayNest

StayNest is a modern, responsive web application, designed to explore and showcase beautiful stays such as beachfront cottages, mountain retreats, and luxury villas. The platform focuses on clean UI, smooth user experience, and attractive property listings.

✨ Features

💰 View price per night for each property

🖼️ High-quality property

📱 Fully responsive design (mobile, tablet, desktop)

🎨 Clean and modern UI inspired by Airbnb

⚡ Fast and lightweight frontend


🔗 Live Website: not available

1.Clone the repository
https://github.com/Akash-kirar/StayNest.git
